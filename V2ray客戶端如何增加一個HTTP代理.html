V2ray客戶端如何增加一個HTTP代理

在我安装了Samba4 、Aria2之后路由器30M空间，只剩下了4M，卸载点东西吧，想到我只有1条科学上网线路，而 Clash 比较适合处理多条线路，所以我决定暂时卸载Clash 用V2ray一段时间

V2ray客戶端如何增加一個HTTP代理

在 OpenWrt 部署了 V2ray 服務，如果要使用它，局域網中的設備就要設置一個 Socks 代理即 ip：port, Windows 和類 Debain 系統都可以輕鬆設置 Socks 代理。

但對與 Android 和 IOS 設備就比較麻煩，它們連接無線的時候，只能設置 HTTP 代理，所以就好解決方案不是在移動設備再安裝 App, 而是讓 V2ray 支持 HTTP 代理，方法也很簡單，只需在 inbound 位置追加一個 HTTP 代理，注意要使用不同的端口。

重啓 V2ray ，移動設備在 WIFI 設置位置設置一個 HTTP 代理，就也可以用 V2ray 的網路了。

OpenWrt

我是比较倾向安装原版V2ray的，但是她的体积太大了。对于新路由3下载 v2ray-linux-mipsle.zip 这个版本，删掉一些无关文件,这些文件还是有32M

那只能试试 upx这个神器了，我的系统是 ElementaryOS,下载后把可执行文件拷贝一下就算安装了

进入v2ray目录
