How to config WireGuard as vpn on Debain/Ubuntu

Today I will show you how to config WireGuard on Debain/Ubuntu.

The first thing we need to do is instalation.
if you use Unbuntu, it’s easy for you according to WireGuard offical website. Just do that.

If you use Debain, you need to add a source to install it.
Under directory ‘/etc/apt/sources.list.d’, make a file like xxx.list and write the content ‘deb http://deb.debian.org/debian buster-backports main’ into the xxx.list file. Mine is ‘a.list’

Secondly, we need to enable network forward

Configure WireGuard

  1. iptables

Find our Network Interfaces

Choose your ip range

Here I choose 10.0.44.0/24 as my range

  1. Generate keys

We need to run it twice because we need two groups.

  1. Make wg0.conf file

Look the template, Interface represents server. Set your server’s ip form your ip range and choose your port. If the Interface use privatekey from group A , the Peer need to use publickey from group B

vi /etc/wireguard/wg0.conf

Now we can start it. Run ‘wg-quick up /etc/wireguard/wg0.conf’, you will see

  1. Configure client

The configuration for client is a little different. We can save its configuration file and import it.
