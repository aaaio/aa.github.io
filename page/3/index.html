Shadowsocks-libev Error

想装 shadowsocks-libev 看能加速一下游戏，有些机场游戏节点要么差要么贵，真不适合我这样的穷人。
用 snap 在两台 Debain 上安装了 shadowsocks-libev，防火墙没问题，ss-server 运行也没问题，但在本地运行 ss-local 就不行一直报错显示 ERROR: remote_recv_cb_recv: Connection reset by peer
连续重装几次系统无果，用 V2ray 来搞 shadowsocks-libev 同样也连不上。搜索一下，有人和我遇到了类似的问题，看了一下作者关于这个问题的回复，说这是网络问题

更改配置 timeout : “6000” 后问题依旧，我猜想是 GFW 在搞鬼吧，不整了

2020.06.20 update
现在换用 go-shadowsocks2 网络恢复正常

JavaScript Comparison operators

Comparison operators JavaScript

JavaScript has both strict and type–converting comparisons. A strict comparison (e.g., ===) is only true if the operands are of the same type and the contents match. The more commonly-used abstract comparison (e.g. ==) converts the operands to the same type before making the comparison. For relational abstract comparisons (e.g., =), the operands are first converted to primitives, then to the same type, before comparison.

安裝了 Hexo 置頂插件，爲了只在留言板啓用評論，我更改評論代碼加入判斷 post.top === ‘1’ 居然一直是 false, 我想它的類型不是 string, 改成 post.top == ‘1’ 就沒有了問題。
