如何使用tinyfecVPN自建游戏加速器

我之前想采用 shadowsocks-libev、udp2raw、udpspeeder来加速游戏，但是 shadowsocks-libev 出了一些问题，又没有钱换新服务器，所以这次尝试 tinyfecVPN + udp2raw 组合来自建游戏加速器。据说，udp2raw、udpspeeder、tinyfecVPN 本意就是用来玩外服游戏的。把整个过程弄一遍，我花了一天的时间，是比较折腾的。如果你有幸看完了教程，但还是不能使用，可以在首页评论区留言，一起探讨一下。

Server

编译无限制版的tinyfecVPN

为了防范树大招风，作者的 relases 只能当普通加速器使用，即只能加速 Server 上的某个服务，想要让它加速游戏,能访问 Server 之外的网络就需要使用无限制版，这需要我们自己编译，下面以我使用的 Debain 9 为例。

完成后，拷贝到了 /root/

下载 udp2raw，放到 /root/

配置

根据作者的模板，修改一下密码，如果愿意的话，也可以修改一下端口。-k 1234 不用修改

现在 Server 端的设置都完成了！

Client

由于 tinyfecVPN 还没有 Windows 版客户端，所以我们只能安装到包括 OpenWrt在内的 Linux系统中。我只在 Windows 10 下玩游戏，动那唯一的路由器有断网的风险，所以我也把她们运行在了虚拟机里。

将 Windows 的网关设置成虚拟机IP,DNS设置为 8.8.8.8

如果没有意外，现在就可以使用了。下面这张图是我在中午的时候，APEX 所显示的延迟。相比之前动辄40%的丢包，现在的效果已经很好了。虽然仍有丢包，但是游戏已经可以愉快地玩了。
