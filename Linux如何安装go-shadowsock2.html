Linux如何安装go-shadowsock2

Introduction

由于 Shadowsocks-libev 运行 ss-local 会一直报错： ERROR: remote_recv_cb_recv: Connection reset by peer ，导致无法使用，这个问题使我纠结了很久，我想不明白问题出在了哪里。难道是我网络的原因吗？可是用手机移动网络依旧无法连接。是服务器的原因吗？ 之前在位于 RU 、US 的两台服务器Debain系统下进行测试都不行，今天我又花大价钱购入 HK 一台服务器，结果还是不能用。所以这个问题的原因不是服务器也不是本地网络。想使用shadowsocks只能暂时切换到其他版本。2020年只剩下 go 版本还在更新。在服务器安装过后，测试可以完美使用。

Server

服务器使用系统为 Debain 10

安装

go-shadowsocks2

运行

只需更改 your-password 、端口8488，后台运行可以使用 screen,然后在新窗口下按 Ctrl + a + d
