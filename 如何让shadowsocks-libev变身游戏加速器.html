Server

shadowsocks-libev

我使用的是 Debain 系统，安装 shadowsocks-libev 比较简单不再展示，由于 SS 协议能被检测出来，所以使用 v2ray-plugin 还是有必要的。

安装 golang

安装 v2ray-plugin

生成证书

curl https://get.acme.sh | sh
~/.acme.sh/acme.sh –issue -d mydomain.me –standalone -k ec-256

编写 shadowsocks-libev 配置文件

vi /etc/shadowsocks-libev/config.json

运行如果出现 read cert > open /root/.acme.sh/xxxx_ecc/fullchain.cer: permission denied， vi /lib/systemd/system/shadowsocks-libev.service 更改用户为 root

加速网络

udp2raw
udpspeeder

下载相应可执行文件

下面以 Linux amd64 平台为例

运行

client

openwrt-shadowsocks
luci-app-shadowsocks ipk
shadowsocks-libev ipk

opkg install iptables-mod-tproxy ipset luci-compat

v2ray-plugin

下载 v2ray-plugin-linux-mips-* 到本地，解压后，选择你需要的版本，新路由3使用的是 v2ray-plugin_linux_mipsle_sf， 重命名为v2ray-plugin上传路由器 /usr/bin目录
